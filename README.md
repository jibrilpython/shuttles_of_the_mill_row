# Shuttles of the Mill Row

A mobile archive for collectors of industrial weaving antiquities — cataloging heavy wooden fly-shuttles, internal pirns, and mechanical bobbins from the age of power-loom weaving.

## Features

- **Shed** — searchable shuttle registry with fiber-type filters
- **Mill Map** — interactive shuttle-flight physics visualization
- **Logbook** — collection telemetry by fiber, loom class, timber, and wear
- **Bench** — side-by-side specimen comparison

## Run

```bash
flutter pub get
flutter run
```
